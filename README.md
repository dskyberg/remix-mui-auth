## Env

The following env values must be set.  Consider using a `.env` file:

```toml
GITHUB_CLIENT_ID=<your id>
GITHUB_CLIENT_SECRET=<your secret>
BASE_URL=http://localhost:3000
SESSION_SECRET=<your session secret>
```

## Install it and run:

```bash
npm install
npm run dev
```
